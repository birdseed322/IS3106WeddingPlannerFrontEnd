#### Some notes
React components do not take in `id='foobar'` as you would expect in normal HTML.
React-Bootstrap components like `<Container>` have id attributes work normally,
however for custom-defined components like `<HeartyNavbar>`, you need to manually specify and apply it using props

Also, React-Bootstrap components have convenient ways to access the default bootstrap themes and styles, using
`bg="primary"` (has other values as well, eg. `light`, `success`) and `variant="light"` attributes.

However, in order to override those themes, you have **remove `bg`** attribute (not sure about `variant`, it mainly changes the font colour) before applying inline-css/class/id selector.
